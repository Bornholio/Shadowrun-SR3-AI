## Stat Line Format

`B Q S C I W E R | Init | Atk`

**B** — Body. `10/4` = second value is armor (ballistic and impact).

**Q** — `base×run_multiplier`. Aquatic = swimming multiplier. Additional multipliers (fly, arboreal, burrow, roll) noted in entry.

**S** — Strength. Default damage codes if no attack listed: claws/dangerous appendages = (STR+2)M; ram/butt without horns = (STR−2)M; otherwise (STR)M.

**C** — Charisma. `—` = non-sentient.

**I** — `base/Perception`. Single value = both. Base: spell resistance, puzzles. Perception: detecting prey/enemies.

**W** — Willpower.

**E** — Essence.
| Notation | Meaning |
|---|---|
| `E6` | Mundane — not dual natured |
| `E(6)` | **Dual natured** — exists simultaneously in physical and astral |
| `E6A` | **Primarily astral** — interacts physically only through powers |
| `E(6)A` | **Primarily astral, dual on manifestation** — becomes dual natured when using Manifestation |
| `E2D6` | Essence determined randomly (vampiric critters) |
| `ESp` | Essence by special rules — see entry |

**R** — Reaction. No asterisk = 1D6 Init dice · `*` = 2D6 · `**` = 3D6.

**Attacks** — "Humanoid" = uses weapons/unarmed as metahuman. Reach shown as `+1 Reach` or `−1 Reach`. Multiple values = multiple attack modes; see entry.

## Size Variation Table

| 2D6 | Size | Attribute Points |
|---|---|---|
| 2 | −50% | −4 |
| 3–4 | −20% | −2 |
| 5–7 | — | — |
| 8–9 | +10% | +2 |
| 10–11 | +20% | +4 |
| 12 | +50% | +8 |

Essence, reach, attack type, movement multipliers, damage modifiers unchanged by size. Exceptions noted per critter.

## Opposition Pools and Professional Rating

Standard NPCs use Professional Rating 1–5 in place of all pools unless specified as major NPCs by GM.
