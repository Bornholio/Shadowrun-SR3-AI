## Load Guide

### `MAPS/transit_south_corridor.md`
The operational core for the campaign's primary travel route. Contains the full station-by-station reference from King Street to Lakewood — security ratings, LS presence, SIN risk, and volcanic damage notes per station. Includes travel times, lahar crossing status, and the chokepoint implications of the White River and Puyallup River bridges. Load this file whenever movement along the Auburn/Sumner/Tacoma corridor is in play.

### `MAPS/transit_network_reference.md`
Network-level overview covering all transit types in the sprawl: rail, bus, monorail, ferry, and private vehicle. Includes SIN risk by network type, the downtown King Street hub connection table, and a district-level security quick reference for runner transit planning. Load this file when the question is which network to use, not which station to use.

### `MAPS/transit_worldbuilding.md`
Named fictional transit services for use as NPC hooks, encounter flavor, and contact texture. Covers ride-hail, courier, and charter categories. Also contains brief volcanic zone transit notes (Orting, Carbanado, Sumner confluence) and the Glow City transit dead zone note. Load this file when a specific service name is needed in play or when fleshing out how goods or people move through a scene.

Smuggler routes — GM notes only, not in any transit file.
Salish-Shidhe border crossing procedures — not yet documented.
Informal Barrens transit (deep Puyallup, Redmond beyond Touristville) — brief footnote in worldbuilding file only; detail is GM-improvised.
