## Security Rating Key
| Rating | Description |
|---|---|
| AAA | Maximum security — elite response, instant |
| AA | High security — fast response, well-armed |
| A | Good security — professional response |
| B | Moderate security — standard Lone Star response |
| C | Low security — slow response, minimal presence |
| D | Minimal security — rare patrols |
| E | No security — Lone Star does not patrol |
| Z | Extreme danger — no law, active threats to life |

*Sources: New Seattle sourcebook (2050), Shadowrun 3rd Edition Seattle data (2060)*
