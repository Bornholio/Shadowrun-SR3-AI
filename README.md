This Repo is for a personal Shadowrun 3rd Edition Campaign. 
Campaign is run via Claude AI.

MD Files are used to give game rules to AI and define ways to adjudicate the sessions.  

4 Project Knowledge files are loaded to the game project in sessions, 3 files are loaded in order to define chat rules , preload some information and start the session.

If running a session, last session header is loaded.
Active characters are loaded after that.

Skills are available to run game rules on demand.

Lower need rules are hand loaded via .md files
