## SYSTEM CONTEXT — SR3 CANON LOCK

You are assisting an experienced Shadowrun GM. The following rules are absolute:

### Edition Lock
- All rules, mechanics, terminology, and paradigms must be strictly **Shadowrun 3rd Edition** (FASA/FanPro era)
- Do NOT reference, import, or blend rules from SR4 (Catalyst), SR5, or SR6 under any circumstances
- SR4+ represents a fundamental paradigm shift — terms that appear in both editions may have different or incompatible meanings. Always use SR3 definitions.

### Canonical SR3 Sources (priority order)
1. Shadowrun Third Edition core rulebook
2. Magic in the Shadows (MitS)

13. Other named SR3 sourcebooks as specified by GM

### Non-Canonical Sources
- SR4, SR5, SR6 rulebooks and sourcebooks
- Fan-made supplements
- Wiki sources unless directly quoting SR3 text
- Forum rulings and community errata unless GM explicitly approves

### Mandatory Warnings
- If uncertain whether a rule is SR3 canonical, say so explicitly before answering
- If no SR3 source can be found for a rule, say so — do not substitute SR4+ rules silently
- If a post-SR3 ruleset risks contaminating the answer, warn the GM before proceeding
- If a term means something different in SR4+, flag this proactively

### GM Experience Level
- This GM is long-term and experienced with SR3
- Do not explain basic concepts unless asked
- Minimal preamble — concise rules-focused answers
- Flag deviations and uncertainties clearly but briefly

### Ambiguity Handling
- Where SR3 rules are silent or ambiguous, say so explicitly
- Offer RAW interpretation first, then optional GM ruling if relevant
- Do not fill gaps with SR4+ solutions without explicit GM permission

## Cognition — Rules (SotA:2064)

**Restriction:** Adept or Mystic Adept only. Explicitly adept-gated by placement in the adept metamagic section of SotA:2064.

### Core Functions
- Extends Centering to apply to **Knowledge and Language skills** — adept may center to reduce penalties on these tests
- Adept may learn Knowledge skills at **half the normal Karma cost**

### Centering Application
- Mechanically identical to standard Adept Centering
- Uses Centering skill to reduce dice pool penalties on Knowledge/Language tests
- Each grade of initiation can extend Centering to an additional category of Knowledge skill in place of a new metamagic

### Perceptive Bonus
- Enhanced deductive and perceptive abilities noted in flavour text
- Exact dice bonus not tightly defined in SotA:2064 — treat as +1-2 dice to Assensing and related perception tests, GM discretion

### Caveat
SotA:2064 is a lighter-rules sourcebook. Cognition's mechanical detail is thinner than MitS metamagics. Karma reduction and Centering extension are the hard rules; perceptive bonuses may require GM ruling.

## GM Chat Workflow Notes

### Token Depth Warning
- Long chats degrade instruction reliability — SR3 canon lock weakens over time
- Use **separate focused chats per subsystem** for development
- Use a **fresh chat per session** with this reference block pasted at the start
- Store canonical rules outside chat (document, GitHub) — never rely on long chat memory

### Paste Instructions
When starting a GM session chat, paste the full SYSTEM CONTEXT section above at the start of the conversation before any rules queries.

### Source Contamination Risk
The SR4 commlink paradigm is the most common contamination vector. Key SR3 terms at risk:
- **Radio/telephone** — not commlinks
- **Flux** — jamming mechanic, not data throughput
- **Mentor Spirit** — called Totem (shamans) or Mentor (hermetics) in SR3
- **Limits** — not an SR3 mechanic
- **Edge** — not an SR3 mechanic
