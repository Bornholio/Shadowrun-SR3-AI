## Core Language Notes
- Context and intonation carry significant meaning — same words differ by register
- Status of speaker vs listener affects word choice throughout
- Sperethiel closely resembles Japanese in its status-sensitivity (SR source†)
- Formal vs informal register is grammatically marked, not just stylistic

## Particles & Prepositions

| Particle | Meaning | Rule |
|---|---|---|
| *ke* / *k'* | "of the" | Elides to *k'* before a vowel |
| *ti* / *t'* | "of" | Elides to *t'* before a vowel |
| *co* / *con* | "to" | Shifts to *con* before a vowel |
| *se-* | positive/beneficial prefix | Attaches directly to root |
| *ver-* | reversal/inversion prefix | *resp* (listen) → *veresp* (reply) |
| *me-* | nominalizer prefix | *daron* (to die) → *medaron* (death) |
| *od* | "and" | Does not elide |
| *a* | "or" | Does not elide |

## Suffixes

| Suffix | Function | Rule |
|---|---|---|
| *-it* / *-nit* | Derisive | Becomes *-nit* after a vowel |
| *-ha* | Emphatic repetition | Attaches as suffix |
| *-tish* | Agent noun — "one who" | Insert *-i-* between consonant and suffix: *telegitish* = keeper |
| *-nagee* | Adverbially "willfully" | Derived from *nagé* |
| *-ya* | Negator | Modifier suffix |
| *-ri* | Interrogative / rhetorical | Terminal suffix |
| *-sa* | Plural nominative | Attaches to noun or suffixed form |

| *li* | Formal | Affix |

## Verb Conjugation †
*Conjugate by affixing to verb root.*

### Indicative Voice

| Person | Singular | Plural |
|---|---|---|
| First person | *-ar* | *-aris* |
| Second person (formal) | *-it* | *-int* |
| Second person (familiar) | *-ist* | *-intas* |
| Third person | *-at* | *-asta* |

**Example — *resp* (to listen):**
*respar* (I listen) / *respat* (he listens) / *resparis* (we listen) / *respasta* (they listen)

### Imperative Voice

| Form | Suffix |
|---|---|
| Singular | *-o* |
| Plural | *-osa* |

**Canonical example:** *telego carronasto* = "keep still" (singular imperative of *teleg*)

### Subjunctive Voice
Three forms; full paradigm not in source:
- **Normative** — could/should/would
- **Speculative** — hypothetical
- **Poetic** — literary/ritual register

## Noun Declension †
*Declension by context, prefix, or both. Rules are guidelines.*

### Singular

| Case | Marker | Notes |
|---|---|---|
| Nominative | root word, no prefix | Default/glossary form |
| Genitive | *ti-* | "of" |
| Dative | *to-* | Long "o" as in "toe"; or by context |
| Accusative | *té-* | Rhymes with "say"; or by context |

### Plural

| Case | Marker | Notes |
|---|---|---|
| Nominative | *-sa* affix | "the [noun]s" |
| Genitive | *si-* | Parallel to singular *ti-* |
| Dative | *so-* | Long "o" as in "toe"; or by context |
| Accusative | *sé-* | Rhymes with "say"; or by context |

## Compound Word Construction
**Pattern: modifier + noun** or **verb-root + agent-suffix**

| Rule | Example | Result |
|---|---|---|
| *se-* + root | *se-* + *seterin* | *se'seterin* = good morning |
| *ver-* + verb | *ver-* + *resp* | *veresp* = to reply |
| *me-* + verb | *me-* + *daron* | *medaron* = death |
| root + *-tish* | *teleg* + *-i-* + *-tish* | *telegitish* = keeper |
| root + *-nit* | *goro* + *-nit* | *goronit* = barbarian |
| noun + noun | *speren* + *thiel* | *sperethiel* = speech/language |
| adj + noun | *sa* + *raegh* | *saraégh* = strict lord |
| noun + *-sa* | *meraerth* + *-sa* | *meraerthsa* = the warriors (plural) |

**Path name formula:** *Mes ti' + [root] + (-tish) + -sa*
- *Mes ti'Meraerthsa* = Path of the Warriors
- *Mes ti'Telenetishsa* = Path of the Scholars (*telene* + *-tish* + *-sa*)

## Key Phoneme Rules for Construction
- Hard *c* only — no soft-c
- Hard *g* only
- *sh* is a distinct phoneme, not a cluster
- *th* is a distinct phoneme — dental fricative
- Accent *é* marks stress on final syllable; default stress is penultimate
- *ll-* initial is valid — lateral approximant
- Elision (*k'*, *t'*) occurs before vowels only

## Language Origins — Research Notes

**Sperethiel source family:** Old Irish / Celtic — not Proto-Celtic, not PIE.
Structural evidence: Old Irish sandhi elision, conjugation suffix pattern, nominalizer prefix, dental fricative *th*, distinct *sh* phoneme. Hard g/c rule is a constructed simplification not present in Old Irish.
**Reference tool:** eDIL (dil.ie) — covers Old Irish c.700–900 CE, which is the closest attested layer to Sperethiel roots.
**Checking method:** Match attested Sperethiel roots against eDIL phonetically, verify semantic alignment. Divergence suggests Proto-Celtic depth or authorial invention.

**ED language family map:**

| Language | Source Family | Notes |
|---|---|---|
| Sperethiel | Old Irish / Celtic | Western branch; authors likely used popular reference |
| Throalic | Semitic pattern | *-im* plural, apostrophe cluster breaks, Hebrew/Arabic feel |
| Scythan | PIE steppe layer | Guttural, archaic; underlayer of Throalic |

**Discard pile:** *de Tuatha*, Celtic Double Cross adventure vocabulary — Irish flavor, not canonical Sperethiel. Tir na nÓg vocabulary should be verified separately before treating as Sperethiel.
