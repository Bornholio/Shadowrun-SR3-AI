# SR3 Action Economy
*Skill file — load on demand*
*Source: SR3, Rigger 3, Matrix, MitS, Critters*

---

## Combat Turn

**Duration: 3 seconds.**
All dice pools refresh at start of each Combat Turn. Unused dice lost at end of turn. Karma Pool refreshes every 24 hours.

---

## Initiative Pass Flow

1. All characters act in descending Initiative order
2. Subtract 10 from all Initiative Scores
3. Score > 0 = acts again · Score ≤ 0 = done for this turn
4. Repeat until no scores remain above 0
5. New Combat Turn — refresh pools, re-roll initiative

---

## Actions Per Combat Phase

| Option | Allocation |
|---|---|
| Standard | 2 Simple Actions |
| Alternative | 1 Complex Action |
| Always available | +1 Free Action (any one Combat Phase) ¹ |

*¹ Multi-Tasking adept power (Singer): 2 Free Actions per Initiative Pass instead of 1.*

**Free Action timing:** Declared at start of Combat Phase. May be used in another character's Combat Phase (resolves last) — only after character's first Combat Phase in the turn.

**Declaring (multiple characters same phase):** Reverse Initiative order — lowest first, highest last.
