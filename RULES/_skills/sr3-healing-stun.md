# SR3 STUN RECOVERY — GM REFERENCE

**Edition Lock: SR3 only.**
*Stun tracking: see sr3-damage-conditions. Stim patch stats: SR3 core p. 305.*

---

## RECOVERY TEST

| Element | Value |
|---|---|
| Dice pool | Body **or** Willpower — whichever is higher |
| Target Number | 2 |
| Modifiers | Apply current Stun **and** Physical wound modifiers |

---

## RECOVERY TIME

- **Base:** 60 minutes per 1 box of Stun.
- **Actual time for 1 box** = 60 minutes ÷ successes rolled.
- After that time elapses, erase **1 box** from the Stun Condition Monitor.
- Repeat the test for each subsequent box.

**Example:** 3 successes → 60 ÷ 3 = 20 minutes per box.

---

## UNCONSCIOUSNESS

A character rendered unconscious by **Deadly Stun** does **not** wake up until Stun is reduced to **Serious** (6 boxes or fewer).

---

## REST REQUIREMENT

- Character must be **completely at rest** throughout recovery.
- If rest is interrupted: recovery stops; roll the Body/Willpower Test again using current condition.
- The new result **can never exceed the first roll**.

---

## TREATMENT LIMITS

| Method | Effect |
|---|---|
| Medical treatment | Does not help |
| Magical spells | No known spell assists |
| **Stim patches** | Temporary workaround — significant drawbacks apply |
