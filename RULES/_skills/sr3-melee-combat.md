# SR3 MELEE COMBAT — GM REFERENCE

**Edition Lock: SR3 only.**

---

## RESOLUTION SEQUENCE

1. Attacker rolls Combat Skill + Combat Pool vs TN 4 + modifiers → successes.
2. Defender rolls Combat Skill + Combat Pool vs TN 4 + modifiers → successes.
3. Higher successes wins. **Tie = attacker wins** at base damage.
4. Winner stages: +1 damage level per 2 net successes. If at Deadly, Power +1 per 2 remaining net successes.
5. Loser resists: Body + Combat Pool vs TN = attacker Power − Impact armor (min 2). Every 2 successes = −1 damage level.

---

## KEY DIFFERENCES FROM RANGED

- Both sides roll Combat Skill (not Dodge)
- Base TN both sides = **4**
- Tie = **attacker wins**
- Armor = **Impact** (not Ballistic)
- Power = **Strength + weapon modifier**
- No separate dodge step — defender skill roll IS the defense
- Full Defense is a special mode (see below)

## MODIFIERS

| Situation | Modifier |
|---|---|
| Called shot | +4 |
| Friends in melee (own side) | −1 per friend, max −4 |
| Friends in melee (opponent's side) | +1 per friend, max +4 |
| Visibility impaired | Visibility Table at **half value, round down** (Full Darkness: full value) |
| Wounded | See sr3-damage-conditions |
| Longer Reach | −1 per point advantage* |
| Inferior Reach | +1 per point disadvantage* |
| Multiple targets | +2 per additional target |
| Superior position | −1 |
| Opponent prone | −2 |

\* Reach modifier applies to **one side only** — the character with longer Reach chooses whether to apply it as −TN to their own attack or +TN to opponent's attack.

---

## REACH

Difference in Reach ratings = modifier amount.  
Character with higher Reach chooses: −TN own attack **or** +TN opponent's attack. Not both.  
Trolls: natural Reach 1, cumulative with weapon Reach.

---

## FULL DEFENSE

Defender chooses defense-only mode. Cannot deal damage even with more successes.

1. Defender rolls Combat Skill only (**no Combat Pool**) vs TN 4 + modifiers.
2. Defender successes > attacker successes → attack blocked.
3. If attacker still has net successes: defender may roll Dodge — **Combat Pool only** vs TN 4 + modifiers. Dodge successes alone exceed attacker net = clean miss.
4. Subtract dodge successes from attacker net. Remaining net successes stage damage up (+1 level per 2).
5. Defender makes Damage Resistance Test as normal. Remaining Combat Pool may be used.

**Monofilament whip:** If attacker misses solely because of Full Defense, attacker rolls Whip Test vs TN 6. Zero successes = attacker struck by own whip (see Weapons table notes).
