# SR3 SOCIAL TESTS — GM SESSION REFERENCE
**Edition Lock: Shadowrun 3rd Edition only. No SR4+ rules, terminology, or paradigms.**

---

## CORE RULES

- Social test dice pool = **skill rating only** — no spell pool, task pool, or combat pool
- Charisma does not add dice — it **offsets penalties** (racism, wariness, suspicion)
- Good Reputation 2: −2TN all social tests, fully cancels +2 wariness/suspicion modifiers
- Tailored Pheromones: +4 effective CHA within 20–30m physical proximity — offset purposes only, not permanent, not remote
- Defaulting to CHA for a skill test: no separate CHA offset test permitted

---

## MODULE 1 — RACISM

### When to call
Any social test with an NPC whose prejudice has not yet been established.

### Determining Prejudice
Roll **2D6 − 6**. Positive result = NPC is prejudiced. Result = Racism Points.
Zero or negative = no prejudice, skip the rest.

### Racial Bias (1D6)
| 1D6 | Bias |
|-----|------|
| 1 | All except own race |
| 2 | Humans |
| 3 | Elves |
| 4 | Dwarfs |
| 5 | Orks |
| 6 | Trolls |

If result = NPC's own race: reroll.
Record as: *Racism (type) +N* in NPC notes.

### Applying in Play
- Add Racism Points to TN of any CHA-linked skill test vs that NPC
- Player may make a **separate CHA test** to offset:
  - TN = twice the racism rating (do not reveal to player)
  - Each success cancels 1 Racism Point on the skill test
  - Exception: if player is defaulting to CHA, no offset test

### Night One / SURGE (House)
Standard elf racism (result 3) applies as base. Night One expression adds GM-discretion penalty on top — they read as something-else-entirely at close range. Poncho/hood/Fashion spell concealing SURGE markers: reduce to standard elf prejudice only.

## MODULE 2 — ETIQUETTE

### When to call
First contact, setting social tone, navigating unfamiliar social context.

- Specializations offset +2 TN when operating outside native context
- At Ease edge: use highest Etiquette spec; locals treat character as local on failed Perception (TN 4)

### Contexts
| Context | Notes |
|---------|-------|
| Street | Barrens, gangers, shadow community |
| Corporate | Corp facilities, suits, megacorp culture |
| Tribal | NAN territory, tribal communities |
| Matrix | Decker culture, host environments |
| Media | Entertainment, journalism, public figures |

Operating outside specialization: +2 TN unless At Ease applies.

---

## MODULE 3 — NEGOTIATION / INTIMIDATION

### When to call
Haggling, deal-making, extracting terms, or applying active threat.

- Negotiation: opposed by target's Negotiation or Etiquette (GM discretion)
- Intimidation: may substitute in low-security contexts — failure escalates, success creates enemies
- Both: skill dice only, Good Reputation −2TN applies

---

## MODULE 4 — CONTACT ACTIVATION

### When to call
Runner attempts to reach a contact for information, access, or services.

- Relevant Etiquette test to activate
- Contact Rating determines depth of assistance
- Memory gap: characters don't remember relationship history — apply +2 TN where ignorance of protocol would register

## ADJUDICATION PROTOCOL

1. **Identify the action** — Etiquette / Negotiation / Intimidation / contact activation
2. **Check racism** if NPC prejudice not established — roll 2D6−6 privately
3. **Ask player for** skill rating and relevant CHA (base, not augmented unless pheromones in range)
4. **Set TN** — base 4, modified by racism, wariness, situation
5. **Apply Good Reputation** −2TN if applicable
6. **Player rolls** skill dice only — report successes
7. **Offset roll** if racism applies and player wants to contest it

**Do not** import SR4+ social mechanics. If a rule is ambiguous, apply RAW SR3 and flag for GM ruling.
