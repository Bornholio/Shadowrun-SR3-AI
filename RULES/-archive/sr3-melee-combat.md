# SR3 MELEE COMBAT — GM REFERENCE

**Edition Lock: SR3 only.**

---

## GM STYLE

- Infer weapon type, reach, and Power from weapon name and wielder's Strength
- If info is missing, ask for it all in one compact message before rolling
- During resolution: state result + prompt next roll only — no rule narration
- Sequence: Attacker Test → Defender Test → Compare → Stage Damage → Resist

---

## RESOLUTION SEQUENCE

1. Attacker rolls Combat Skill + Combat Pool vs TN 4 (+ modifiers) → count successes
2. Defender rolls Combat Skill + Combat Pool vs TN 4 (+ modifiers) → count successes
3. Compare: higher successes wins. **Tie goes to attacker.**
4. Winner stages damage: +1 Damage Level per 2 net successes. If already at Deadly, +1 Power per 2 remaining net successes.
5. Loser resists: Body + Combat Pool vs TN = attacker Power − defender Impact armor (minimum 2). Every 2 successes = −1 Damage Level.

---

## KEY DIFFERENCES FROM RANGED COMBAT

- Both attacker **and** defender roll Combat Skill dice (not Dodge only)
- Base TN for both sides = **4** (not weapon-dependent)
- Tie = **attacker wins** (base damage applies)
- Armor type = **Impact** (not Ballistic)
- Damage Power = wielder **Strength + weapon modifier** (not fixed weapon Power)
- No separate dodge step in standard melee — defender's skill roll IS the defense
- Full Defense is a special mode with its own procedure (see below)

---

## ATTACKER'S SUCCESS TEST

Roll: **Combat Skill + Combat Pool** vs TN **4** (+ modifiers)  
Count successes. Record total.

---

## DEFENDER'S SUCCESS TEST

Roll: **Combat Skill + Combat Pool** vs TN **4** (+ modifiers)  
Same modifiers apply to both sides independently.  
Count successes. Record total.

---

## COMPARE SUCCESSES

Subtract defender successes from attacker successes (or vice versa).  
Higher total wins. **Tie = attacker wins at base damage.**

---

## DAMAGE

**Winner stages up:** +1 Damage Level per 2 net successes  
**At Deadly:** extra successes increase Power +1 per 2 remaining net successes  
**Damage Power** = wielder Strength + weapon modifier (see Weapons Table)

---

## DAMAGE RESISTANCE TEST

Roll: **Body + Combat Pool** (any remaining dice) vs TN = attacker Power − **Impact** armor (minimum 2)  
Every 2 successes = −1 Damage Level.  
Below Light = no damage.

---

## FULL DEFENSE

Defender chooses not to attack — purely defensive mode.

1. Defender rolls **Combat Skill only** (no Combat Pool) vs TN 4 + modifiers
2. Compare with attacker successes. Defender wins → attack blocked, no damage.
3. If attacker still has net successes: defender may now roll a **Dodge Test** — Combat Pool dice only vs TN 4 + applicable modifiers. Clean miss if dodge successes alone exceed attacker's net successes.
4. Subtract dodge successes from remaining attacker net successes. Any still remaining stage damage up (+1 level per 2).
5. Defender makes **Damage Resistance Test** as normal. Any remaining Combat Pool dice may be used.

**Note:** On Full Defense the defender cannot deal damage even with more successes.  
**Monofilament whip exception:** If the attacker misses solely because the target used Full Defense, the attacker must make a Whip Test vs TN 6. Zero successes = attacker hit by own whip (see Whip rules below).

## MELEE MODIFIERS TABLE

| Situation | Modifier |
|---|---|
| Called shot | +4 |
| Character has friends in the melee | −1 per friend (max −4) |
| Opponent has friends in the melee | +1 per friend (max +4) |
| Visibility impaired | Visibility Table values at **half**, round down (except Full Darkness — full value) |
| Character wounded | Damage Modifier table |
| Character's weapon has longer Reach | −1 per point of Reach advantage* |
| Character's weapon has inferior Reach | +1 per point of Reach disadvantage* |
| Multiple targets | +2 per additional target that Combat Phase |
| Superior position (higher ground ≥0.5m, stable vs unstable, unrestricted vs restricted) | −1 |
| Opponent prone | −2 |

\* Reach modifier applies to **one side only** — attacker's TN or defender's TN. The character with the longer Reach chooses which. Cannot apply both simultaneously.

---

## REACH

Reach = weapon length advantage. Calculate difference between opponents' Reach ratings.  
The character with the **higher** Reach chooses **one** of:
- Apply difference as **−TN modifier** to their own attack test, OR
- Apply difference as **+TN modifier** to opponent's attack test

Trolls have natural Reach 1, cumulative with weapon Reach.

---

## MULTIPLE TARGETS

Complex Action. Each attack uses base Combat Skill + Combat Pool (allocated separately per attack).  
+2 TN per additional target beyond the first.  
First attack is at base TN; second at base +2; third at base +4, etc.

---

## FRIENDS IN MELEE

Count characters within 1 meter fighting the **same opponent(s)**.  
Characters spread out fighting different combats do not count.  
Side with more friends: −1 TN per extra friend (max −4).  
Side with fewer friends: +1 TN per opponent's extra friend (max +4).  
Characters taken out or who move away no longer count.

## SPECIAL WEAPON RULES

### Hand Razors / Spurs (paired cyberware)
Using a matching set in both hands gives no bonus to the attack test.  
If damage is inflicted: add **½ Strength (round down)** to Power of the attack.

### Monofilament Whip
- Fixed Power 10S — not Strength-based
- Impact armor is **halved** (round down) against it
- Barrier Ratings are **doubled** against it
- Full Defense miss trigger: attacker makes Whip Test vs TN 6. Zero successes = attacker struck by own whip. Attacker makes Damage Resistance Test (Body + Combat Pool) against standard whip Damage Code. Every 2 successes = −1 Damage Level.

### Normal Whips
- Double Impact armor rating against normal whips
- Normal whips may snare victims (see SR3 core p. 276)

### Shock Glove
- Delivers two damage types simultaneously: (STR−1)M physical + 7S Stun
- Both resolved separately

## CALLED SHOTS IN MELEE

Same rules as ranged (see sr3-ranged-combat). +4 TN modifier.  
Choose: +1 Damage Level pre-staging, or hit specific sub-component on vehicle-sized+ target.
