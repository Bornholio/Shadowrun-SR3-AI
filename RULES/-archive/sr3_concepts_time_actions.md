# SR3 CORE CONCEPTS — TIME & ACTIONS

**Edition Lock: SR3 only.**

---

## COMBAT TURNS

- Each Combat Turn ≈ 3 seconds
- Characters act in order of Initiative, fastest first
- Multiple **Initiative Passes** possible per turn
- Each character acts once per pass (their **Combat Phase**)
- Pass structure: highest acts first, subtract 10 per pass, ≤0 = no action

---

## ACTIONS PER COMBAT PHASE

| Type | Allowance |
|---|---|
| Complex Action | 1 per phase |
| Simple Action | 2 per phase (or instead of 1 CA) |
| Free Action | 1 per phase (own or another's) |

---

## DELAYED ACTIONS

- Declare delay instead of acting
- May take Free Actions while waiting
- Intervene at any later phase in the same turn or next pass
- While delaying: no other actions

---

## MOVEMENT

- Total movement allowance divided across number of passes taken
- Walk = QUI × 1m/turn | Run = QUI × 3m/turn
- Divided per pass (not per turn)
