# SR3 CORE CONCEPTS — INITIATIVE

**Edition Lock: SR3 only.**

---

## INITIATIVE FORMULA

`Reaction (augmented) + Initiative Dice rolled`

- Base: 1d6. Cyberware/magic may add more.
- Dice listed as `1d6 (2d6)` = 2 dice total (1 base + 1 bonus)
- **Not a test** — Rule of Six does not apply

---

## INITIATIVE BY MODE

| Mode | Reaction Base | Dice |
|---|---|---|
| Physical | (QUI + INT) / 2 + mods | 1d6 + bonuses |
| Rigging (VCR) | VCR bonus only | Per VCR rating |
| Driving with datajack | Physical REA +1 | Standard |
| Decking (Matrix) | Response Increase only | Per deck |
| Astral projection | INT | 1d6 + 20 bonus |

**Matrix:** Only Response Increase modifies REA/Initiative. No other mods except injury.
**Rigging:** Only VCR modifies. No other mods except injury.
**Astral:** REA = INT. +20 Initiative bonus. 1d6 dice.

---

## INITIATIVE PASSES

1. Roll initiative (all characters)
2. Highest acts first → subtract 10 → next pass
3. Character at ≤0 = no more actions this turn
4. Repeat until all characters are at ≤0
