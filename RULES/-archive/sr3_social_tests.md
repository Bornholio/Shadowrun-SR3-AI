# SR3 Social Tests — GM Reference
*Malice Family Campaign*
*File location: RULES/sr3_social_tests.md*

---

## Core Principles

- Social test dice pool = **skill rating only** — no spell pool, task pool, or combat pool
- Charisma does not add dice — it **offsets penalties** (racism, wariness, suspicion)
- Good Reputation 2: −2TN on all social tests, fully cancels +2 wariness/suspicion modifiers
- Tailored Pheromones: +4 effective Charisma within 20–30m for offset purposes only — not permanent, not remote
- Defaulting to Charisma for a skill test: character may NOT make a separate Charisma test to offset racism

## Racism

### Determining NPC Prejudice
Roll 2D6 − 6. If result is positive, NPC is prejudiced.
Result = **Racism Points** — higher = more racist.

*Example: roll of 9 → 9−6 = 3 Racism Points*

If result is 0 or negative: NPC has no racial prejudice.

### Racism Table
| 1D6 | Racial Bias |
|-----|-------------|
| 1 | All except own race |
| 2 | Humans |
| 3 | Elves |
| 4 | Dwarfs |
| 5 | Orks |
| 6 | Trolls |

If result = NPC's own race: reroll until two distinct biases indicated or result is "all races."

Record as: *Racism (type) +N* in NPC notes.
*Example: Racism (Orks) +3*

### Applying Racism in Play
- Add NPC's Racism Points against character's metatype to TN of any Charisma-linked skill test
- Character may make a **separate Charisma Test** to offset:
  - TN = twice the NPC's racism rating (GM only — do not reveal)
  - Each success counteracts 1 Racism Point
  - Charisma test successes offset racism only — they do not help the skill test directly
- **Exception:** If character is defaulting to Charisma for the skill test, no separate Charisma test permitted

### Night One / SURGE Modifier (House)
Standard elf racism (result 3) applies as base. Night One expression stacks additional GM-discretion penalty — they read as something-else-entirely at close range, not merely unusual elf. Poncho/hood/Fashion spell concealing SURGE markers reduces this to standard elf prejudice only.

## Etiquette

- Primary social skill for first contact, setting tone, navigating social context
- Specializations function as lingos — offset +2 TN penalty when operating outside native social context
- At Ease edge: use highest Etiquette specialization, locals treat character as local on failed Perception (4)

### Etiquette Contexts
| Context | Notes |
|---------|-------|
| Street | Barrens, gangers, shadow community |
| Corporate | Corp facilities, suits, megacorp culture |
| Tribal | NAN territory, tribal communities |
| Matrix | Decker culture, host environments |
| Media | Entertainment, journalism, public figures |

Operating outside your Etiquette specialization: +2 TN unless At Ease edge applies.

---

## Negotiation

- Used for haggling, deal-making, extracting favorable terms
- Opposed by target's Negotiation or Etiquette (GM discretion)
- Good Reputation −2TN applies
- No pool augmentation — skill dice only

---

## Intimidation

- Active threat or implied consequence
- May substitute for Etiquette/Negotiation in low-security contexts
- Risk: failure escalates situation, success may create enemies
- Charisma offset for racism still applies

---

## Instruction / Leadership

- Instruction: teaching a skill or conveying complex information
- Leadership: directing group action under pressure
- Pool = skill only, TN varies by complexity and group size

---

## Contact Activation

- Contacts activated via relevant Etiquette test
- Rating determines depth of assistance available
- Memory gap penalty: characters don't remember relationship history — GM may apply +2 TN for contacts where ignorance of protocol is likely to register
