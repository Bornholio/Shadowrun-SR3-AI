# Small Unit Tactics — GM Reference
*SR3 · Shadowrun Third Edition core*
*3PR Campaign Reference — Singer primary user*

---

## Core Mechanic

**Linked stat:** Intelligence
**Default:** Intelligence
**Specializations:** BattleTac Systems, Vehicle Tactics, Matrix Tactics

Used to convey tactical analysis to teammates, granting Initiative bonuses or Combat Pool dice next Combat Turn.

---

## Action Cost

| User type | Action to convey orders |
|---|---|
| Standard (no BattleTac) | Complex Action |
| BattleTac master unit (Singer) | **Simple Action** |
| Self only (no communication needed) | No action |

---

## Singer — Effective SUT

| Source | Value |
|---|---|
| Base SUT skill | 5 |
| TC bonus (Rating 4) | +4 |
| **Effective SUT** | **9** |

**Max teammates boosted** = base SUT skill rating = **5**
*(TC bonus adds to skill for tests but does not increase the target cap)*

---

## Executing SUT — Sequence

1. Singer spends Simple Action to convey orders (last action of Combat Turn)
2. One SUT test made — Singer rolls effective SUT 9 vs TN per target
3. Every 2 successes vs TN = +1 Initiative to that team member next Combat Turn
4. Initiative bonus may not exceed target's Reaction (cap per character below)
5. Optional: convert Initiative bonus to Combat Pool dice instead (1:1)

---

## Multiple SUT Users

If more than one character runs SUT for the same target:
- Only highest bonus applies, **−1** for conflicting orders
- Example: +3 from one source, +5 from another → target receives **+4**

---

## Optional — Combat Pool Conversion

Each +1 Initiative bonus may instead be taken as +1 Combat Pool die for next Combat Turn.
Player's choice per character receiving the bonus.

---

## Ambush / Surprise Use

SUT used in preparation for ambush (by either side):
- Adds **1 die per 2 successes** to Reaction/Surprise Tests
- Applies to ambushers or ambushees

---

## SUT and TC Surprise Pool

Singer's TC at Rating 4: **100% of Combat Pool usable for Surprise Tests.**
SUT ambush bonus stacks with TC surprise pool — both apply independently.
