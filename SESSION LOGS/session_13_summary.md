- Great Form Invoking confirmed procedure: standard conjuring → Great Form test vs (Force×2)−Grade → Drain vs same TN
- Astral signature erasing: Conjuring skill for spirit signatures, Sorcery for spell signatures, vs TN = Force, Complex Actions = Force
- Cleansing: Sorcery vs BGC×2, reduces BGC by 1 per 2 successes, Drain = [BGC level]D
- Stealth Open Test high die + Invisibility bonus = TN for perception tests against invisible character
- Multistrike: +1 TN per additional target, −1 Power per additional target
- Confusion: WIL test TN = Force to act each pass; −6 TN to all tasks while affected
