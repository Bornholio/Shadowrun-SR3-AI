## Overview

GT threat team files are split by escalation tier. Load only the tier relevant to current play.
Do not load higher tiers speculatively — they contain spoiler-level GT posture data.

Each tier file is self-contained with full stat blocks for that tier's additions.
Retained operatives (carry-overs from prior tiers) are noted with a pointer to their source file.

---

## Load Triggers

### GT_threat_tier1_grey_shutter.md
**Load when:**
- Family gets close to the HMS second signatory name
- Family pushes on Harmon (Meridian's contact, VESTIBULE) and GT becomes aware
- Reyes activates surveillance response
- Any question about who might be watching the family from the GT side

**Contains:** GS-1 through GS-4 full stat blocks. Design notes. Tactical posture.

---

### GT_threat_tier2_cold_notch.md
**Load when:**
- Grey Shutter is burned or withdrawn
- Family makes contact with Okonkwo
- Harmon goes public with HMS signatory intel
- Reyes's new node is located by the family
- CN-4 auto-escalation trigger is reached (she assenses the family and hits masked auras)

**Contains:** CN-1 through CN-4 full stat blocks. GS-1 and GS-4 referenced by pointer only (load Tier 1 file if blocks needed). Auto-escalation note for CN-4.

---

### GT_threat_tier3_iron_sill.md
**Load when:**
- Family contacts Okonkwo directly
- Signatory name goes public
- CN-4 reports unreadable auras (auto-trigger)
- GT principal has authorized hard measures
- Combat or direct action against the family is imminent

**Contains:** IS-1, IS-2/3/4, IS-5, IS-6 full stat blocks. CN-3 and CN-4 referenced by pointer only (load Tier 2 file if blocks needed). Full GM tactical notes including BattleTac gap, magical ceiling, escalation brake, and If Iron Sill Fails.
