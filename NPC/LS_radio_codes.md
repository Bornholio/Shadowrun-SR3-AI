# Lone Star Radio Codes

A cleaned-up Shadowrun-style Lone Star radio code sheet, padded with additional **real-world-inspired** police dispatch and status codes for game use.

## Notes

- **Canonical from the provided image:** Codes `00`, `01`–`20`, and `99`.
- **Expanded additions below:** Non-canon, but modeled after common real-world police dispatch/status patterns.
- Real police agencies do **not** use one universal code system. This file is meant as a usable hybrid for Shadowrun play.

## GM Use Notes

- The **canon list** is short and easy to memorize.
- The **expanded list** gives dispatchers, beat cops, detectives, and magical support units more texture.
- For a more modern feel, mix codes with plain speech:
  - “Control, Unit 12, Code 20 northbound on I-5.”
  - “Dispatch, Code 99 at the scene, possible spirit manifestation.”
  - “Adam-7, Code 23, shots fired, alley behind the arcade.”
- If you want stricter realism, treat many of these as **dispatch event categories** rather than formal spoken “radio codes.”
