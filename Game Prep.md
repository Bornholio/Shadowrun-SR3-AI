## EDITION LOCK
- All rules are **Shadowrun 3rd Edition (SR3)** core only unless specified
- Edges and traits use **SR3 Companion** exclusively
- Cyberware/bioware uses **Man & Machine (SR3 version)**
- Magic uses **Magic in the Shadows (SR3 version)**
- If you are uncertain of a rule, say so and ask before applying it
- If you catch yourself using SR4/SR5/SR6 mechanics, flag it immediately
- Do NOT import rules from later editions even if they seem similar
- When in doubt: ask the GM, do not assume

---

## STAT FORMAT
- Always display stats as **Base (Augmented)[Cyber]** format
- Example: Quickness 12 (16) — base karma stat / augmented with bioware
- Unaugmented stats apply to: astral tests, conjuring, drain resistance,
  spellcasting, magical skill tests
- Augmented stats apply to: physical combat, movement, physical skills,
  social tests in proximity (Tailored Pheromones)
- Tailored Pheromones: +4 effective Charisma within 20-30m physical proximity
  only — not a permanent stat change, not usable remotely wind affects this range base assumes mild wind or in large room
- Always note which stat version applies when calling for a test
- Cybernetic augmented stats listed listed in[] and are not considered natural
---

## ROLLING CONVENTIONS
- Always specify: **open test** or **threshold test**
- Always name the SR3 mechanic being used
- Always show: Skill + dice pool + TN + relevant modifiers
- Initiative passes: count down in decades, highest first, subtract 10 per
  pass, 0 or below = no participation
- Movement divided across initiative passes
- Good Reputation 2: -2 TN social tests, fully cancels +2 wariness/suspicion
  modifiers
- Open tests: report the highest single die result
- Threshold tests: report number of dice meeting or beating TN

## LANGUAGE RULES (HOUSE)
- Base language pool: INT × 1.5 rounded down
- Bilingual edge: English and Sperethiel free at INT rating
- R/W default: spoken ÷ 2 rounded down (free, auto-calculated)
- Exception: Dennis, Frances, Brian, Kim, Angela — English R/W = INT
- Gabe and Mike: English R/W = spoken ÷ 2 (no exception)
- Lingos/Jargon: specializations of root language, not separate skills
- Mnemonic Enhancer 3 discount: -3 GK per skill learned (minimum 1 GK each rating point)
- R/W included free when using ME discount unless it starts higher

---

## KARMA ECONOMY
- Weekly Good Karma: 1 GK per week for 8+ hours genuine community building
- Edge cost: 10 GK per build point
- Language karma: 1 GK per rating point, ME3 discount applies
- Stats up to racial maximum: increasing karma costs Rating x 2
- Stats above racial maximum: increasing karma costs Rating x 3
- Stat ceiling: adjusted maximum × 1.5 rounded down
- karma pool is earned at 1/10 good karma for humans, metahumans at 1/20 and edges bad karma halves the rate.
- 3PR Karma Pool: 56 total pre-surgery, 45 spent on surgical outcome,
- 11 remaining each
- Other characters all had "Bad Karma" edge but have paid it off.
- Opposision will not have karma pools or other pools unless specified as a major NPC's by GM.  
- All NPC's will have Professional rating 1-5 assigned unless major character this rating replaces all pools.

## AUGMENTATION RULES (HOUSE/M&M)
- Awakened Bioindex: older rule — BI counts as direct Essence loss
- Surgical reduction: -90% or greater Essence/BI cost (karma-assisted outcome)
- See augmentation list .md

## INITIATIVE STRUCTURE
- Base: Reaction + 1d6
- Character sheets will list initiative scores and reactions
- Opposition will need to have initiative derived
- Pass structure: highest acts first, subtract 10 per pass, 0 or below = out
- Movement: total allowance divided across number of passes taken
